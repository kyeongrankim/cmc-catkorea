CMC-CAT Korea 홈페이지 Version 2 업로드 방법

1. GitHub 저장소 접속
   https://github.com/kyeongrankim/cmc-catkorea

2. 기존 index.html을 새 파일로 교체하거나, 전체 파일 업로드
   - Code 화면에서 Add file 또는 + 버튼 → Upload files
   - 이 폴더 안의 index.html, css, js, images, gallery, videos 폴더를 모두 드래그해서 업로드
   - Commit changes 클릭

3. 1~3분 뒤 홈페이지 확인
   https://kyeongrankim.github.io/cmc-catkorea/?v=2

4. 버튼 문구/연락처 수정 위치
   - index.html 파일 안에서 Contact, Education Application, hero 영역을 검색해서 수정

주의:
- videos/hero.mp4 파일은 메인 배경 영상입니다.
- 영상 파일 업로드가 실패하면 videos 폴더는 빼고 먼저 업로드해도 됩니다. 그 경우 메인 사진이 대신 보입니다.
